Installing the Template

  • → After unzip the Template Files, you will found HTML folder with all this files you will upload to your site.
  • → If you want to see this Template in any web broswer just open them, you dont need internet conextion.
  • → Start your FTP client (I persoanlly use Filezila which is free) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.
  • → Locate the domain folder you wish to upload your Template files to on your server (usually inside the www folder). You are going to want to click and drag every file that you have labeled .html, .css, .php, .js into your FTP client.
  • → Once the files are done uploading, rename the file you want to be your home page as 'index.php'.
  • → It should looks like: www.yourwebsitename.com/index.html

I'm not going to tell you about when i discovered this great tool, or necessarily why i like it and so on. I'm actually going to push you into using it :) I mean it, you can't be a serious developer unless you're using it.

Scenarios:

  • → You want to change the size/color/etc of a certain font and search into 3 stylesheets with 1000 lines each and you can find that damn rule which contains the property
  • → You want to get rid of an ugly space somewhere
  • → Some block has some absolute position and overlays another block
  • → You want to inspect an element to see if it has some sort of margin padding etc.

Mainly (for me), Firebug is about tracking the css code and to debug a website seeing the actual final (generated code). Font Icon Edition

Go to Fontello and select the family font you want to use. In this case we use: Font Awesome and Entypo.

Download the zip file with all the fonts.

With this template we already downloaded 565 fonts (so you dont have to).

Unzip the downloaded file: you will see fonts+css styleshets.

Now go to the Template font folder and replace it with the new fonts and the new css stylesheet.

If you want to use a different icon from Font Awesome and Entypo, you don´t have to download anything, just change the name of the icon (i class)

To change one icon for another, go to Fontello select the icon you want to use, copy the name of the icon and paste it in your html file.
